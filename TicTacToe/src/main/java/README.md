## Задача: "Крестики-нолики"

### Описание
Возьмём за основу игру крестики-нолики 3x3.

### Пример вывода при запуске main
```
ДЕМОНСТРАЦИЯ

X O -
O X - 
O - X
ПОБЕДИЛИ КРЕСТИКИ

X X X 
O O -
O - - 
ПОБЕДИЛИ КРЕСТИКИ

O - X 
O X - 
O - X 
ПОБЕДИЛИ НОЛИКИ

X O X 
O X O 
O X O 
НИКТО НЕ ПОБЕДИЛ
```
